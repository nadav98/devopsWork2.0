# פרויקט ניווט דפים (Front-End)

## תיאור הפרויקט
אתר בסיסי הכולל שלושה דפי HTML המקושרים ביניהם. הפרויקט מדגים שימוש בקישורים, כפתורים ופונקציונליות JavaScript לשינוי עיצוב ה-DOM.

## מבנה הקבצים
כל הקבצים מופיעים תחת התיקייה front.
- `index.html`: דף הבית עם כותרת "Hello WORLD".
- `secondPage.html`: דף ביניים המקשר להמשך התוכן.
- `thirdPage.html`: דף חדשות ועדכונים עם כותרות מעוצבות.
- `script.js`: קובץ לוגיקה המכיל פונקציה לשינוי צבע הכותרת (`highlightHeadline`).

## איך להריץ
1. פתחו את הקובץ `index.html` בדפדפן.
2. השתמשו בכפתורים המופיעים בכל דף כדי לעבור לדף הבא או לחזור לדף הבית.

## טכנולוגיות
- HTML5
- JavaScript (DOM Manipulation)
